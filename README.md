# 🚀 My Portfolio

A modern web application built with **React**, **TypeScript**, and **Vite**, styled using **Tailwind CSS** and **shadcn-ui**.

---

## 📖 Overview

This project is designed with a focus on speed, scalability, and developer experience.  
It includes a flexible component system, clean architecture, and support for rapid prototyping.

---

## 🛠 Tech Stack

- ⚡ [Vite](https://vitejs.dev/) – Fast build tool
- ⚛️ [React](https://react.dev/) – UI library
- 🟦 [TypeScript](https://www.typescriptlang.org/) – Type-safe JavaScript
- 🎨 [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework
- 🧩 [shadcn/ui](https://ui.shadcn.com/) – Accessible component library

---

## 📦 Installation

Make sure you have **Node.js** and **npm** (or **pnpm/yarn**) installed.  
Clone the repository and install dependencies:

```sh
# Clone the repo
git clone <YOUR_GIT_URL>

# Enter project folder
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## 🚧 Development

- Run the dev server:  
  ```sh
  npm run dev
  ```
- Build for production:  
  ```sh
  npm run build
  ```
- Preview production build:  
  ```sh
  npm run preview
  ```

---

## 🌍 Deployment

This project can be deployed on platforms like **Vercel**, **Netlify**, or **Cloudflare Pages**.  

For example, with Vercel:

```sh
npm run build
vercel deploy
```
